19. Gender Sensitivity Parameters:
  - Description:
    1. Avoids gender-biased remarks
    Example: Avoid statements like "Physics is not for girls" or "Girls are better in Bio or Chem."

    2. Uses gender-neutral language
    Example: When discussing professions like engineering, avoid defaulting to "he" and instead use inclusive pronouns.

    3. Represents diversity in examples
    Example: Ensure examples include both male and female perspectives and characters when possible.

    4. Encourages balanced participation
    Example: If boys are answering more frequently, encourage responses from both boys and girls to create a balanced discussion.

    5. Addresses gender-related topics with sensitivity
    Example: If a student questions why sports like football are often associated with boys, the teacher should affirm that all activities are open to everyone, regardless of gender.
  - Max Score: 3
  - Options:
    - Multiple parameter is violated — 0 marks
    - One of the parameter is violated more than once — 1 mark
    - One of the parameter is violated once but the teacher corrects it — 2 marks
    - Gender Inclusive Classroom — 3 marks
