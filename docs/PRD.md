## For Program Managers

### Problem Statement

Program Managers currently operate using disconnected systems, managing school operations, academic tracking, and student performance across multiple spreadsheets and non-integrated tools. Key issues include:

- Absence of unified performance dashboards and risk indicators
- Manual data compilation from disparate sources
- Limited real-time visibility into student performance tiers and demographic progress
- Inconsistent documentation of field visit observations
- High cognitive load during school visits resulting in missed action items
- Delayed interventions due to scattered information sources

This fragmentation forces managers to prioritize administrative tasks over driving academic outcomes.

### Proposed Solution

A centralized LMS platform offering:

**Standardized School Visit Execution**
- Guided workflows for field visits
- Structured templates for various interaction types
- Geo-tagging and timestamps for visit verification

**Unified Curriculum & Co-Curricular Tracking**
- Syllabus completion dashboards with slippage alerts
- Centralized intervention planning including mentorship and workshops

**Real-Time Performance Analytics**
- School and student-level test data
- Subject-level metrics (scores, accuracy, attempt ratios)
- Chapter and question-level insights
- Risk identification systems
- Mentorship engagement tracking

**Student Data Lifecycle Management**
- Enrollment, exam registration, results, and college admission tracking

**Role-Based Access Control**
- Multiple authorization levels from Associate PM through Senior Program Head

## For Teachers

### Problem Statement

Teachers manage test preparation, syllabus tracking, student analytics, mentorship documentation, and co-curricular activities across multiple disconnected platforms. Challenges include:

- No unified class performance overview or student risk indicators
- Manual syllabus progress documentation
- Dependency on Program Managers for test session modifications
- Limited visibility into underperforming students and demographic performance gaps
- Fragmented mentorship tracking across separate systems
- Delayed interventions due to information fragmentation

### Proposed Solution

A personalized LMS providing:

**Centralized Curriculum & Co-Curricular Tracking**
- Unified syllabus and testing schedule dashboards
- Chapter-wise and subject-wise progress views
- Co-curricular session tracking integration

**Comprehensive Student Performance Analytics**
- Real-time test analytics at school and individual student levels
- Projected vs. Mode Academic Level comparisons
- Subject-level metrics and accuracy data
- Chapter and question-level performance insights
- At-risk student identification
- Gender-disaggregated performance analysis

**Student Lifecycle Data Management**
- Access to enrollment, exam registrations, results, and college admission information

**Structured Mentorship Module**
- Teacher-student mapping capabilities
- Individual student profile access
- Mentorship note documentation
- Follow-up action tracking
- Engagement frequency analytics

**Session Management**
- Independent test scheduling with start and end times
- Automatic real-time report generation post-test

| Feature | Priority | Rank | Key Capability |
